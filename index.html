Presentation

Tatami is a twitter-like application, for internal use inside a company.

Tatami is made with the following technologies :

  • Apache Cassandra
  • The Spring Framework
  • HTML5 and Twitter Bootstrap

Tatami is developped by Ippon Technologies

Installation

  • Install Maven 3
  • Run Cassandra from Maven : mvn cassandra:run
  • Run Jetty from Maven : mvn jetty:run
  • Connect to the application at http://127.0.0.1:8080

The default users are "jdubois/password" and "tescolan/password", you can check or modify the Spring Security configuration at /META-INF/spring/applicationContext-security.xml
